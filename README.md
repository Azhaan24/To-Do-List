# To-Do List App
A productivity-focused Android application developed using Java and Android Studio that helps users manage daily tasks efficiently through a simple and interactive interface.

Features: Add new tasks, Edit existing tasks, Delete tasks, Task management interface, Responsive UI design, Easy navigation

Tech Stack: Java, Android Studio, XML

Purpose: This project was built to strengthen Android development concepts including UI handling, data management, RecyclerView usage, and user interaction design.
